## Projeto para Portifólio

O projeto é uma Landing Page para uma marca de relógios masculinos de luxo. O produto tem como inspiração a força da natureza em geral, mesclada com a tecnologia, trazendo estética, força e conceito.

Vamos apresentar a página principal do e-commerce, o lançamento, uma breve apresentação da marca, seus produtos e um chamado para ação para que seja possível comprar o relógio que está sendo lançado.

Queremos transmitir a essência da marca em seu visual.

O seu público-alvo são pessoas com espírito jovem, aventureiras e que gostam de ter esse contato com a natureza.

...

## Estudos

Tenho como objetivo criar a landing page com os conhecimentos que adquiri até aqui, usando HTML e CSS puro, sem o auxílio de IA para sua estruturação, escrevendo desde o começo, me obrigando a pesquisar como fazer tal coisa, aprendendo como de fato fazer e aplicando diretamente.

### Primeiro Passo 

A estrutura do site é simples. 
Fiz um html facil de compreender e respeitando sua semantica. 
Usei Sections no Header e no Footer porque não quero que acompanhem a pagina no seu scroll, mas dei IDs para identificalos. 

### Segundo Passo

A estilização do site vai tentar transmitir a essencia da marca. 
Comecei com confg gerais definindo cores e fontes. Fonte utilizada vai ser a Poppins, e as cores vao ser tons de terra (Marrom Barro, Madeira e Marfin para contraste)  
Estou tentando seguir um fluxo de semantica no CSS respeitando cada section. Iniciando pelo Header.
